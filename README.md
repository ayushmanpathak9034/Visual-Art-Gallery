# 🎨 Collage of Colors – Art Gallery Website

## 📌 Project Overview

**Collage of Colors** is a simple and visually appealing **Art Gallery website** developed using basic HTML.
The project showcases different types of digital artwork along with descriptions, sources, and pricing details.

This project was created as part of a **college assignment** to demonstrate fundamental web development concepts.

---

## 🚀 Features

* 🖼️ Display of multiple artworks (Flowers, Skeleton, Autumn, Desktop, Psychedelic)
* 📝 Detailed description of each artwork
* 🔗 External source links for images
* 📊 Structured table for artwork pricing
* 📋 Lists showing resources used and platforms for submission
